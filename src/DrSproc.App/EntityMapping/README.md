The following read-type methods are avialable for Custom Mapping:
 - ```ReadString()```
 - ```ReadLong()```
 - ```ReadNullableLong()```
 - ```ReadInt()```
 - ```ReadNullableInt()```
 - ```ReadShort()```
 - ```ReadNullableShort()```
 - ```ReadByte()```
 - ```ReadNullableByte()```
 - ```ReadDouble()```
 - ```ReadNullableDouble()```
 - ```ReadDecimal()```
 - ```ReadNullableDecimal()```
 - ```ReadBoolean()```
 - ```ReadNullableBoolean()```
 - ```ReadDateTime()```
 - ```ReadNullableDateTime()```
 - ```ReadGuid()```
 - ```ReadNullableGuid()```
